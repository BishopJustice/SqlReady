SqlReady.com

Making Work Easier

SqlReady.com is a collection of operational tools that uses columns copied from spreadsheets to make tedius office tasks easier. From formatting lists for SQL queries to opening multiple web pages at a time, SqlReady.com allows you to complete everyday taks faster than ever.

The first step in almost all of these tools is to copy a column of text from a spreadsheet and paste it into the input box.

    SqlReady

    Formats spreadsheet columns for lookup in an sql statement

    Phone Cleaner

    Cleans up columns of phone numbers to be just numbers

    Tab Opener

    Allows you to open multiple links at once
